codes in this folder are written for the analyses of Kir2.1 data for the Rocchi et al paper.
requires the preprocessed .mat files. 

How to use:
run the Main.m routine for each animal separately, at the beginning of the routine, it will ask you to go to the animal folder and select the traces you want for analyses. It will saves the results in the current directory under a folder named results
After finishing, run the stats_check.m, it will ask you for the folder that results are and asks you to select them, then it visualizes the results with statistical assessment
